# Component | Text Area

Render a text area input box.

<Tabs groupId="sdkLanguage">
    <TabItem value="node" label="TypeScript / JavaScript">
    ```js
    ui.textArea("text-area-key", { 
        label: "Write your story...",
        onEnter: (value) => console.log(value)
    })
    ```
    </TabItem>
    <TabItem value="python" label="Python">
    ```python
    ui.text_area(
        "text-area-key",
        label="Write your story...",
        on_enter=lambda value: print(value)
    )
    ```
    </TabItem>
</Tabs>
