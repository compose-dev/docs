# Component | JSON Input

:::note
Introduced in SDK version 0.26.3
:::

Render a JSON input with:

- **Syntax highlighting and formatting** for the input data
- **Error checking** to ensure well-formed JSON
- **Line numbers** for better readability

<IFrame type="components-input-json" heightPx={450} />

<Tabs groupId="sdkLanguage">
    <TabItem value="node" label="TypeScript / JavaScript">
    ```js
    ui.jsonInput("user-data", {
        initialValue: { users: [ /* ... */ ] }, // Replace with actual JSON data.
        onEnter: (value) => db.updateUserMetadata(value)
    })
    ```
    </TabItem>
    <TabItem value="python" label="Python">
    ```python
    ui.json_input(
        "user-data",
        initial_value={ "users": [] }  # Replace with actual JSON data.
        on_enter=lambda value: db.update_user_metadata(value)
    )
    ```
    </TabItem>
</Tabs>

Learn more about [working with inputs](/guides/forms).

## Change input height

You can change the height of the input element via the `style` property. For example, to render the input with a height of 800px:

<Tabs groupId="sdkLanguage">
    <TabItem value="node" label="TypeScript / JavaScript">
    ```js
    ui.jsonInput("user-data", {
        style: { height: "800px" }
    })
    ```
    </TabItem>
    <TabItem value="python" label="Python">
    ```python
    ui.json_input(
        "user-data",
        style={"height": "800px"}
    )
    ```
    </TabItem>
</Tabs>

## API reference

### Function signature

<Tabs groupId="sdkLanguage">
    <TabItem value="node" label="TypeScript / JavaScript">
    ```js
    type ValidJson =
    | string
    | number
    | boolean
    | null
    | ValidJson[]
    | { [key: string]: ValidJson };

    ui.jsonInput(
        id: string,
        properties?: Partial<{
            label: string,
            description: string,
            required: boolean,
            initialValue: ValidJson,
            validate: (value: ValidJson) => string | void,
            onEnter: (value: ValidJson) => void,
            style: Style
        }>
    )
    ```
    </TabItem>
    <TabItem value="python" label="Python">
    ```python
    JsonPrimitive = Union[str, int, float, bool, None, Mapping[str, Any], List[Any]]
    ValidJson = Union[Mapping[str, JsonPrimitive], List[JsonPrimitive], JsonPrimitive]

    ui.json_input(
        id: str,
        *,
        label: str | None = None,
        description: str | None = None,
        required: bool = True,
        initial_value: ValidJson = None,
        validate: Callable[[ValidJson], str | None] | None = None,
        on_enter: Callable[[ValidJson], None] | None = None,
        style: Style | None = None
    )
    ```
    </TabItem>
</Tabs>
