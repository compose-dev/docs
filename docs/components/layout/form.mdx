# Component | Form

:::tip

Looking to build forms in your app? Read our [forms guide](/guides/forms) for a comprehensive overview with examples and best practices on how to build forms. This page serves as an API reference for the `ui.form` component.

:::

Form is a generic container component that can be used to group other input components.

It includes all the functionality of a [stack](/components/layout/stack), while also providing additional utilities for building forms.

This page serves as an API reference for the `ui.form` component. If you're looking for a guide on how to build forms, check out the [forms guide](/guides/forms).
