# Component | Spinner

:::tip

Looking to show loading states in your app? We recommend using the [`page.loading()` method](/page-actions/loading) for the vast majority of use cases. It's far simpler to use and comes with additional features such as disabling user interactions while loading.

The `spinner` component is useful for complex UI situations when you need more control over exactly where the spinner is displayed.

:::

Render a loading spinner.
