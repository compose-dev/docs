# Component | Image

Render an image.

<Tabs groupId="sdkLanguage">
    <TabItem value="node" label="TypeScript / JavaScript">
    ```js
    ui.image("https://example.com/image.png", { style: { width: "600px" } })
    ```
    </TabItem>
    <TabItem value="python" label="Python">
    ```python
    ui.image("https://example.com/image.png", style={"width": "600px"})
    ```
    </TabItem>
</Tabs>

<IFrame type="components-display-image" heightPx={550} />

At this time, the image component does not support local images. You must pass a URL to an image hosted on the internet.

## API reference

### Function signature

<Tabs groupId="sdkLanguage">
    <TabItem value="node" label="TypeScript / JavaScript">
    ```js
    ui.image(
        src: string,
        options: Partial<{
            style: Style
        }>
    )
    ```
    </TabItem>
    <TabItem value="python" label="Python">
    ```python
    ui.image(
        src: str,
        *,
        style: Style | None = None
    )
    ```
    </TabItem>
</Tabs>
