# Page Action | Log

Use `page.log()` to log information to audit logs. Only available on Pro plans.

<Tabs groupId="sdkLanguage">
    <TabItem value="node" label="TypeScript / JavaScript">
    ```js
    page.log("Deleted database record in user's table", {
        severity: "info",
        data: {
            userId: 123,
            recordId: 456,
        },
    });
    ```
    </TabItem>
    <TabItem value="python" label="Python">
    ```python
    page.log(
        "Deleted database record in user's table",
        severity="info",
        data={
            "user_id": 123,
            "record_id": 456,
        },
    )
    ```
    </TabItem>
</Tabs>

## Audit Logs

### Collecting logs

By default, Compose will log everytime a user runs an app. But, due to Compose's tight security model, it's not possible for us to see what a user does within an app.

For all other logs you may want, it's simple to use `page.log()` to record additional information. All logs are automatically enriched with the user's session, timestamp, the app route, and more. Note that Compose may enforce rate limits to prevent abuse. We'll work with you to increase this limit if needed.

### Viewing logs

Logs can be viewed, filtered, and searched via the Compose dashboard.

### Exporting logs

Logs can be exported to a CSV format. You can also contact us to set up a custom export to a separate service such as S3, Datadog, etc.

## API reference

### Function signature

<Tabs groupId="sdkLanguage">
    <TabItem value="node" label="TypeScript / JavaScript">
    ```js
    page.log(
        message: string,
        properties?: Partial<{
            severity: "trace" | "debug" | "info" | "warn" | "error" | "fatal",
            data: Record<string, any>
        }>
    ): void
    ```
    </TabItem>
    <TabItem value="python" label="Python">
    ```python
    page.log(
        message: str,
        *,
        severity: Literal["trace", "debug", "info", "warn", "error", "fatal"] = "info",
        data: Dict[str, Any] = {}
    ) -> None
    ```
    </TabItem>
</Tabs>
